To make genomic data exchange future-ready, CSIRO developed the Serverless Beacon (sBeacon). This innovation utilises a new cloud-architecture paradigm, serverless, or Function-as-a-Service that allows for the seamless scaling of compute resources on public cloud providers. sBeacon provides a solution that is more scalable and economical than the traditional implementation, hence catering to the cohort sizes of the future.
Beacon UI enables the visual interpretation of results and faster querying of the Beacon V2 API with greater ease. Beacon UI is capable of connecting to any V2 compliant Beacon API and provide querying capabilities right away. For added security, username (email) and password based authentication is provided using AWS Cognito as the userpool manager. If you wish to deploy your own Beacon UI instance, please get in touch with us.
